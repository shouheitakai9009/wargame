# 巨大なファットコントローラーをリファクタリングする

## 必ず実施すること

- context7 mcp を使い React のベストプラクティス、アンチパターンを見て当てはまる箇所を修正する
- context7 mcp を使い Redux のベストプラクティス、アンチパターンを見て当てはまる箇所を修正する
- context7 mcp を使い tailwindcss のベストプラクティス、アンチパターンを見て当てはまる箇所を修正する
- context7 mcp を使い shadcn/ui のベストプラクティス、アンチパターンを見て当てはまる箇所を修正する
- useAppSelector 内で計算できるなら必ず計算した結果を返す
- ビジネスロジックは redux にまとめたいが、redux が絡まない場合はカスタムフックかユーティリティ関数等に切り出す
- ReactNode を返す関数を作らない（コンポーネントに切り出すことが正しい）
- map でループしている HTML はコンポーネントとして切り出す
